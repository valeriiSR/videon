Модели контроллеров видеостен

  • VIDEON START (V3)

    Флагман среди контроллеров видеостен, предназначенный для управления большими массивами модулей (до 56 шт.) в составе идеостены.

  • VIDEON STANDART

    Высокопроизводительный процессор, подходящий для инсталляций с видеостенами до 24 дисплеев

  • VIDEON PRO

    Модель видеоконтроллера, предназначенная для инсталляций с видеостенами размером до 36 модулей

  • VIDEON MAX

    Флагман среди контроллеров видеостен, предназначенный для управления большими массива ми модулей (до 56 шт.) в составе видеостены.

Центр обучения Видеон

Тренинг предназначен для инженеров, проектировщиков и
технологов аудио и видео оборудования для бизнеса.
По завершению уения участники тренинга научатся:

  • Проектировать системы видеоотображения (видеостены)
    на базе контроллеров для видеостен

  • Запускать и настраивать контроллер видеостен Videon

  • Осуществлять работу с программным обеспечением, управлять
    контроллером Videon
